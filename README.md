# Mirofish Trading Bot - Claude Fable 5

A sophisticated automated trading dashboard for Polymarket with advanced probability analysis and real-time visualizations.

## Features

- **Real-time Trading Dashboard**: Live P&L tracking, win rates, and trade statistics
- **Probability Lattice**: Visualizes trade outcomes through ball-drop simulation
- **Tail Probability Ridge**: Identifies extreme outcome probabilities
- **Relationship Graph**: Market correlation analysis with network visualization
- **Automated Trading**: Tail sniper strategy with risk management
- **WebSocket Integration**: Real-time updates from backend
